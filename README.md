
Project Overview & Architecture Summary

Executive Summary
• Custom Level Editor with real-time validation and in-editor preview.
• Deterministic grid-based item collection (no physics dependency).
• Addressables-integrated object pooling with concurrent load protection.
• Session-based lifecycle management for clean level restarts.
• Dependency Injection driven modular architecture (Zenject).

Technology Stack

-Unity 6 LTS
-Zenject — Dependency Injection
-UniTask — Async/Await handling
-DOTween — Animation & Pathing
-Addressables — Asset Management
-Odin Inspector — Editor tooling
